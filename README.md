# RunIt Pitch Deck

Interactive pitch deck for **RunIt** — a campus logistics platform connecting time-starved university students with peer runners for errands like food, groceries, pharmacy, printing, and laundry.

Built for the S-VCG 2025 application at the University of Lagos (CMUL).

## Stack

- **Runtime**: Bun
- **Server**: TypeScript (server-rendered HTML)
- **Frontend**: HTMX, vanilla CSS/JS
- **Slides**: Scroll-snap with animated counters, phone mockups, and reveal transitions

## Run locally

```bash
bun install
bun run dev
```

Opens at `http://localhost:3002`.
